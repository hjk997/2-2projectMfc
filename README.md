# IOT Simulator
사물인터넷 시대에 접어들며, 일반인들이 생소하게 느낄 수 있는 iot 기술에 대한 simulator를 제작하였습니다. 
simulator를 통해서 하나의 기기로 일반 가정집의 전자제품을 조종하는 상황을 체험할 수 있습니다. 

## 개발 기간
2018.11.07 ~ 2018.12.11


## 사용 도구 
C++, MFC, Visual Studio

## 구분 
팀 프로젝트 

## 담당 부분
*서버-클라이언트 통신 구현
*전등 기능 구현
*TV 기능 구현 
*냉장고 기능 구현

*전등 사용 모습
'서버' 프로그램에서 전등 켜기/ 전등 끄기 버튼을 누르면 위와 같이 작동합니다. 

![image](https://user-images.githubusercontent.com/56015030/95677478-c20b8400-0c00-11eb-9b53-b161a2db2f0b.png)

*tv 사용 모습
tv 켜기를 누르고 채널을 바꾸면 tv를 켤 수 있습니다. tv 끄기 버튼을 누르면 tv가 꺼집니다. 

![image](https://user-images.githubusercontent.com/56015030/95677517-fa12c700-0c00-11eb-8ed6-ffb3827248e4.png)

*냉장고 사용 모습
사용자가 사용하는 조종 기기에서 냉장고 안에 있는 음식을 확인할 수 있으며, 추가, 삭제가 가능합니다. 집에서도 냉장고를 클릭하면 '현재 냉장고에 들어있는 것들'을 모달창으로 보여줍니다.
